# Roamly Python Backend

Python/FastAPI conversion of the original Node.js/Express backend.

## Stack
- **FastAPI** — replaces Express
- **Beanie + Motor** — async MongoDB ODM (replaces Mongoose)
- **PyJWT** — JWT auth (replaces jsonwebtoken)
- **bcrypt** — password hashing (replaces bcryptjs)
- **google-generativeai** — Gemini AI (replaces @google/generative-ai)
- **httpx** — async HTTP client (replaces axios)
- **slowapi** — rate limiting (replaces express-rate-limit)

## Setup

```bash
cd backend
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env            # fill in your keys
python main.py
```

Server runs at: https://roamly-travel-iota.vercel.app/
